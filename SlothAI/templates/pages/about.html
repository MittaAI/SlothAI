Welcome

I'm really glad you are here and I want to assure you that this site never shares user's information.

21 years ago, I started a search engine I called Grub. Grub's crawler was Open Source and intended to be distributed and somewhat aware about what was changing on the Web, such as fluctuations in the stock market, trends in news or the a price of an item for sale online. Now we have machine learning to help with document evaluation and transformation, I decided to reboot the big dream and build something new from scratch.

The first part of this journey was researching and developing Mitta.us, a service I built that allows you to save and search for documents, including websites. Mitta crawls pages VISUALLY and uses LLMs to transform and expand upon data it collects from you and the Internet. One of the things left to do for Mitta.us was to build out a service for user configurable pipelines. This is that service.

MittaAI is a pipeline management system for document and data processing. MittaAI currently allows for building complex AI pipelines using Python-based templates, which makes it easy to write and run code in the template itself. The system is designed to run calls to AI models (or anything really) asynchronously, which means some types of workloads may be built and run in parallel, lowering inference times considerably for complex workflows.

I borrowed a lot of ideas about how to build both of Mitta's UIs from Loggly, which I founded after leaving Splunk back in 2009. At that time, we were intensely focused on making it easier to deal with logs in a powerfully simple UI. Here, I've applied those ideas to building fast pipelines for transforming data and saving documents and webpages you find interesting.

Mitta is a bootstrapped, sole founder startup. I couldn't have done any this without the support of my family and friends. Be sure to follow or check on the latest status updates @mittacorp or @campbellkord.

Kord Campbell
